# Data Engineer Case - DataPeople Corp.

Building a scalable and modern solution capable of integrating data from multiple sources in a cloud environment, ensuring governance, traceability, and information quality. This solution will be the first step toward structuring the company’s analytical ecosystem, enabling the future use of dashboards, predictive models, and advanced analytics on DataPeople Corp’s workforce.
